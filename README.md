# activiti-bpmn
activiti-bpmn demo，主要用于测试bpmn配置不当是rce。

在实战中如果遇到可以编辑并部署流程图的都可以实现rce。前端传递bpmn流程图可以有两种形式，xml和json格式。

看具体情况，使用Xml2Json将xml转成Json格式；使用Json2Xml将json转换为xml格式。
